DESAFIO ESTÁGIO DATA ENGINEERING

Autor: Arthur Migotto Goulart

Necessário possuir java no computador.

Executar o arquivo DesafioEstagio.jar

Inserir os locais dos arquivos .tsv nos campos do menu, exemplo: C:\Users\budi\Desktop\teste de estágio\deals.tsv

Os arquivos .tsv devem estar padronizados assim:


Contratos :  1° linha descartada por possuir o nome dos campos, 1° coluna o ID do contrato, 2° coluna data do contrato, 3° coluna ID do contato, 4° coluna ID da empresa



Contatos : 1° linha descartada por possuir o nome dos campos, 1° coluna o ID do contato, 2° coluna o nome do contato, 3° coluna a data que o contato foi criado, 4° coluna nome de quem criou o contato,

5° coluna email do contato, 6° coluna telefone do contato, 7° empregado do contato, 8° coluna ID do empregado, 9° coluna o endereço do contato, 10° coluna coordenadas do contato, 11° coluna relação, 12° coluna responsável do contato



Empresas : 1° linha descartada por possuir o nome dos campos, 1° coluna ID da empresa, 2° coluna nome da empresa, 3° coluna a data que a empresa foi criada, 4° coluna nome de quem criou a empresa,

5° coluna email da empresa, 6° coluna telefone da empresa, 7° coluna ID do empregado, 8° coluna nome do empregado, 9° coluna usuário responsável, 10° coluna o ID do setor



Setores: 1° linha descartada por possuir o nome dos campos, 1° coluna ID do setor, 2° coluna o nome do setor



Após isso os dados serão processados e será possível gerar os outputs.

Gerar output venda total por mes: Clique no botao criar arquivo venda total por mês, e insira o desejável nome do arquivo e o ano, exemplo: vendasPorMesNoAno2018 no primeiro campo e 2018 no segundo campo.

Gerar output venda por contato: Clique no botão "criar arquivo venda por contato", e insira o desejável nome do arquivo, exemplo: vendaPorContato

Gerar output proporcao do setor na venda do mês: Clique no botão "criar arquivo proporcao de venda por setor", e insira o desejável nome do arquivo e o mês,

exemplo: proporcaoVendaDoSetorAbril2018 no primeiro campo e 4/2018 no segundo campo.

Todos os outputs serão gerados em formato txt. no padrão csv.

Todos os outputs serão gerados na pasta em que o arquivo DesafioEstagio.jar se localiza.
